# SnowConvert AI - How to get an access code[¶](#snowconvert-ai-how-to-get-an-access-code "Link to this heading")

## Access code information[¶](#access-code-information "Link to this heading")

An access code is required to use the application and is valid for 3 months. You can request an access code directly from the app, and it will only be issued to users with a company domain email address. Once you have an access code, it allows you to convert from any available source language to Snowflake.

## How to Get an Access Code[¶](#how-to-get-an-access-code "Link to this heading")

There are two ways to get an access code in the SnowConvert AI application:

1. **Using ‘Get an Access Code’ Link**:   
   On the ‘Project Creation’ page, you can find the ‘Get an Access Code’ link next to the access code dropdown field. Clicking this link opens a form where you can provide the required information to request an access code, which will be sent to you upon completion.  
   \

   ![Request access code from link](../../../../../../_images/output2.gif)
2. **Using Help Menu**:  
   Once SnowConvert AI is running, you can go to the main bar menu of SnowConvert AI and click on ‘Get an access code’ option which will open the form to request an access code.\

   ![Request access code from help menu](../../../../../../_images/output3.gif)

Note

**NOTE:** Once the form information is sent, you will receive an email with the access code. If you need special assistance, please get in touch with our customer support team at [snowconvert-support@snowflake.com](mailto:snowconvert-support%40snowflake.com).
