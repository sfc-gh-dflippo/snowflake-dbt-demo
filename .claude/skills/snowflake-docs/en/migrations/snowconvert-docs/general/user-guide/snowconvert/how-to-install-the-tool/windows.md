# SnowConvert AI - Windows[¶](#snowconvert-ai-windows "Link to this heading")

## Windows Installation[¶](#windows-installation "Link to this heading")

1. Click on the [downloaded](../../../getting-started/download-and-access) .exe file.
2. That will start the installation process on your computer.

![SnowConvert AI Installation on Windows](../../../../../../_images/image%2827%29.png)

3. Once the process is finished you can open SnowConvert AI from your applications menu.

![Open SnowConvert AI tool](../../../../../../_images/image%28349%29.png)

4. After you start the application you can create a new assessment or conversion project or open an existing one.

![image](../../../../../../_images/image%28463%29.png)

### Setting up the CLI[¶](#setting-up-the-cli "Link to this heading")

1. The first step to set up the SnowConvert AI CLI in your Windows environment is to download the corresponding *.zip* file (see the image below) that contains all the necessary elements to run the CLI version of SnowConvert.\

   ![image](../../../../../../_images/image%28419%29.png)
2. Then you have to extract that data and put it in a convenient location that you can use to include it in your PATH environment variable. In this example, the file is in the C:\ drive. You have to copy the path where your files are located. For this example it would be: `C:\SnowConvert-CLI-arm646-windows`

![image](../../../../../../_images/image%28420%29.png)

4. Search “environment variables” in the Windows search bar and click on **Edit environment variables for your account**.

![image](../../../../../../_images/spaces_-MBBGScp6ufnp_l6ZtP3_uploads_DLOxt1Nb6pi4zJiWZu5u_image%282%29.png)

5. This will open a new window. Click on the **Edit** button while selecting the **Path** variable for your user.
6. This will open another window. Click on **New** and then paste the path that you had copied earlier. You must append this text at the end of the path: **\orchestrator** . For example, if you copied: `C:\SnowConvert-CLI-arm646-windows`, after appending that text, you will have: `C:\SnowConvert-CLI-arm646-windows\orchestrator` .
7. Click on OK to leave the current window, and then click on OK to leave the **Edit environment variables for your account** window.
8. You can now use the **snowct** command in the command line (you might need to open a new command line).
9. Verify that it was added correctly by executing`snowct --version`. You will see the information for your SnowConvert AI CLI version displayed.

   ![image](../../../../../../_images/image%28421%29.png)
