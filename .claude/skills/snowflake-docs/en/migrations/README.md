# Snowflake Migration Tools[¶](#snowflake-migration-tools "Link to this heading")

Snowflake offers two powerful migration tools to help organizations modernize their data platforms:

## [SnowConvert AI](snowconvert-docs/overview)[¶](#snowconvert-ai "Link to this heading")

Converts SQL code from legacy platforms to Snowflake SQL
Supports migrations from:

* Teradata
* Oracle
* SQL Server
* Redshift
* Azure Synapse
* Sybase IQ
* Spark SQL
* Databricks SQL
* BigQuery
* PostgreSQL & Based Languages
* Vertica
* Hive

Enables automated SQL translation to accelerate database migrations

## [Snowpark Migration Accelerator (SMA)](sma-docs/README)[¶](#snowpark-migration-accelerator-sma "Link to this heading")

Converts code from Spark to Snowpark API
Supports:

* Python
* Scala
* Java

Helps modernize data processing applications to run natively in Snowflake

### Both tools offer:[¶](#both-tools-offer "Link to this heading")

* Easy-to-use interfaces
* Automated code conversion
* Accelerated migration timelines
* Path to leverage Snowflake’s modern data platform capabilities
* For more information:

SnowConvert AI: Email [snowconvert-info@snowflake.com](mailto:snowconvert-info%40snowflake.com)

SMA: Email [sma-info@snowflake.com](mailto:sma-info%40snowflake.com)
