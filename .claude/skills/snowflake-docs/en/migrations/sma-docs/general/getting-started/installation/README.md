# Snowpark Migration Accelerator: Installation[¶](#snowpark-migration-accelerator-installation "Link to this heading")

When you [download the Snowpark Migration Accelerator (SMA)](../download-and-access), you’ll receive an installer package. The installation process varies depending on your operating system, as SMA runs locally on your machine. You can install SMA on Windows, MacOS, or Linux. The following pages provide step-by-step installation guides for each operating system.

## Installation Guides[¶](#installation-guides "Link to this heading")

Before installing, verify that your system meets the [requirements](../download-and-access.html#system-requirements) listed on the download page.

Follow the appropriate installation guide based on your operating system.

* [Windows Installation guide](windows-installation)
* [MacOS Installation guide](macos-installation)
* [Linux Installation guide](linux-installation)

After installing the Snowpark Migration Accelerator (SMA), please refer to the [user guide](../../../user-guide/overview) for detailed instructions on using the application.

Note

When your installed version of the Snowpark Migration Accelerator needs updating, you’ll see an “Updates Available” notification in the top right corner. To get the latest version:

1. Click “**UPDATE NOW**” to download the update
2. Follow the installation prompt that appears after the download completes

You can also update by using the [Check for updates option](../../../user-guide/project-overview/configuration-and-settings.html#updating-the-application) in the menu.
