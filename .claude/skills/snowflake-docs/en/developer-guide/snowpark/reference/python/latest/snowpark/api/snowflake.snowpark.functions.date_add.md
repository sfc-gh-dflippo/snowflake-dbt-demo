# snowflake.snowpark.functions.date\_add[¶](#snowflake-snowpark-functions-date-add "Permalink to this heading")

snowflake.snowpark.functions.date\_add(*col: Union[[Column](snowflake.snowpark.Column.html#snowflake.snowpark.Column "snowflake.snowpark.column.Column"), str]*, *num\_of\_days: Union[[Column](snowflake.snowpark.Column.html#snowflake.snowpark.Column "snowflake.snowpark.column.Column"), str, int]*)[[source]](https://github.com/snowflakedb/snowpark-python/blob/v1.44.0/src/snowflake/snowpark/functions.py#L5548-L5585)[¶](#snowflake.snowpark.functions.date_add "Permalink to this definition")
:   Adds a number of days to a date column.

    Parameters:
    :   * **col** – The column to add to.
        * **num\_of\_days** – The number of days to add.

    Example:

    ```
    >>> from snowflake.snowpark.functions import date_add, to_date
    >>> df = session.createDataFrame([("1976-01-06")], ["date"])
    >>> df = df.withColumn("date", to_date("date"))
    >>> res = df.withColumn("date", date_add("date", 4)).show()
    --------------
    |"DATE"      |
    --------------
    |1976-01-10  |
    --------------
    ```
